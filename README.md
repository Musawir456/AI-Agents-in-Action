# AI Agents in Action – Workshop

This repo contains hands-on Google Colab notebooks for building and running AI Agents in real-world scenarios.

## Requirements

Before running the notebooks, you’ll need API keys:

* **Serper API Key** → [Get it here](https://serper.dev/dashboard)
* **Gemini API Key** → [Get it here](https://aistudio.google.com/app/apikey)

## How to Use

1. Clone or open the repo in Colab:
   👉 [GitHub Repo](https://github.com/Musawir456)
2. Open a notebook in **Google Colab** (recommended).
3. Add your API keys in the side panel.
4. Run the cells and start experimenting with AI Agents.
